Listas

Sintaxis

ol>ul(ol)*x

ol = ordered list(lista ordenada)

ul = unordered list (lista desordenada)

li = list item (listado de item)


dl>dt(dd)*x

dl = definition list (lista de definicion)

dt = definition term (definicion de termino)

dd = definition description (definicionde descripcion)

Listas Desordenadas

Son listas que no estan enumeradas

  • Claudio
  • Luciana
  • Mario
  • Javier
  • Gabriel

Listas Ordenadas

Son listas con numeración

  1. Hockey
  2. Futbol
  3. Basquetbol
  4. Tenis
  5. Natación

Listas de Definición

Son listas que parten con un término y en la siguiente linea una definición de este término

Chile (Término)
Pais de residencia (Definición)
Volcanes Hockey
Club de Hockey en Linea
