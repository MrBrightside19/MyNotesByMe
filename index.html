Tips

De h1 hasta h6 son los titulos decrecientes

hr = linea separadora

br = Salto de linea

a = hipervinculos, href es el link

img = es para imagenes

div = contenedores (rectangulos) que toma todo el ancho disponible

flex = son contenedores que pueden variar su forma

class = son para darle atributos a los elementos, como al div

id = es para darle un valor o una etiqueta al elemento

i = ?

u = Subrayar elemento

link = definir acceso a un archivo de la carpeta, CDN, framework u otro
