Tablas con Bootstrap

Sintaxis

class="table" --> Le da la forma de un table desde el framework Bootstrap

class="table-bordered" --> se le agregan bordes a la tablas, con 'borderless se le quitan'

class="table-striped" --> Colores alternados en filas

class="table-hover" --> Cambia el color cuando el mouse flota sobre las filas

class="table-dark" --> Tabla con paleta de colores oscuras

class="table-primary" --> Destaca una fila o celda en particular. Puede ser ademas 'secondary', 'warning', 'danger', 'info', 'success', etc

class="table-sm" --> Hace el table mas pequeño
